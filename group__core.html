typedef unsigned long int GrB_Index

Type of index objects used in OpenGraphBLAS.

It define by :

typedef unsigned long int GrB_Index;

The range of valid values for a variable of type GrB_Index is [0, GrB_INDEX_MAX] where the largest index value permissible is defined with a macro, GrB_INDEX_MAX.

◆ GrB_finalize()

void GrB_finalize ( )

OpenGrapblas finalization function.

An instance of the GraphBLAS C API implementation as seen by an application. An application can have only one context between the start and end of the application. A context begins with the first thread that calls GrB_init and ends with the first thread to call GrB_finalize. It is an error for GrB_init or GrB_finalize to be called more than one time within an application.
