Detailed Description

This structure record some messages used in SpVSpM

Parameters
Indices_NZRows: Indices of the non-zero rows of the matrix selected by the input vector. @capacity_vector_nnz elements should be allocated to it.
LL_Index_Ptr: The indices of all selected non-zero rows in @ListNodes, which is similar to the ptr array in CSC or CSR format. (@capacity_vector_nnz + 1) elements should be allocated to it.
ListNodes: The array which stores all the linked list generated by all non-zero rows/cols. @capacity_nnz elements should be allocated to it.
ListPtr: Array of pointers to all linked tables, which corresponds to LL_Index_Ptr. @capacity_nnz_row elements should be allocated to it.
