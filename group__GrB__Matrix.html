GrB_Matrix_build

Store elements from tuples into a matrix

Store BOOL elements from tuples into a matrix

Parameters
[in,out]CAn existing Matrix object to store the result.
[in]IPointer to an array of row indices.
[in]JPointer to an array of column indices.
[in]XPointer to an array of scalars of a type that is compatible with the domain of matrix, C.
[in]nvalsThe number of entries contained in each array (the same for row_indices, col_indices and values).
[in]dupAn associative and commutative binary operator to apply when duplicate values for the sam location are present in the input arrays.
Returns
