Detailed Description

Perform a reduction across the elements of an object.

These perform a reduction across a GrB_Matrix or GrB_Vector to produce a GrB_Vector or scalar.

Reduce all stored values into a single INT32 scalar.

Parameters
[in,out]cPointer to scalar to store final reduced value into. On input, the scalar provides a value that may be accumulated (optionally) with the result of the reduction operation. On output, this scalar holds the results of the operation.
[in]accumAn optional binary operator used for accumulating entries into an existing scalar c value. If assignment rather than accumulation is desired, GrB_BINARYOP_NULL should be specified.
[in]monoidThe monoid used in the reduction operation. The reduction operator must be commutative and associative; otherwise, the outcome of the operation is undefined.
[in]AThe OpenGraphBLAS matrix on which reduction will be performed.
descAn optional operation descriptor. If the description is not needed to assist, GrB_DESC_NULL should be specified.
Returns
GrB_Info
