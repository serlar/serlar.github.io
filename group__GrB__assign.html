Detailed Description

Assign the contents of a subset of a matrix or vector.

Assign values from one scalar, one GrB_Vector or one GrB_Matrix to a subset of a vector or a matrix as specified by a set of indices. The size of the input vector/matrix is the same size as the index array provided.

Assign the contents a vector to a subset of elements in one row of a matrix. Note that since the output cannot be transposed, a different variant of assign is provided to assign to a column of a matrix.

Parameters
[in,out]CAn existing OpenGraphBLAS Matrix. On input, the matrix provides values that may be accumulated with the result of the assign operation. On output, this matrix holds the results of the operation.
[in]maskAn optional "write" mask that controls which results from this operation are stored into the output matrix C. The mask dimensions must match those of the matrix C
[in]accumAn optional binary operator used for accumulating entries into existing w entries.
[in]uThe OpenGraphBLAS vector whose contents are assigned to (a subset of) a row of C.
[in]row_indexThe index of the row in C to assign. Must be in the range [0, nrows(C)).
[in]col_indicesPointer to the ordered set (array) of indices corresponding to the locations in the specified row of C that are to be assigned. If all elements of the row in C are to be assigned in order from index 0 to ncols - 1, then GrB_ALL should be specified.
[in]ncolsThe number of values in col_indices array. Must be equal to size(u).
[in]descAn optional operation descriptor. If the description is not needed to assist, GrB_DESC_NULL should be specified.
